The Reactjs order quotes system allows for the efficient quoting of HP products for office use based on volume. The system considers private stock consumption and offers customization options for orders.

**Characteristics:**

The user enters the quoting system and selects the type of HP product that he wishes to quote.
The system asks for the desired quantity and the user enters the required quantity.
The system displays the customization options available for the selected product, and the user can select the desired options.
The system calculates the total price of the quote based on the selected quantity and customization, and displays the result to the user.
The user can confirm the quote and proceed to the payment process, or they can make adjustments to the quote again if they wish.
Once the quote is confirmed, the system registers the order and sends it to the factory for production and shipment.
